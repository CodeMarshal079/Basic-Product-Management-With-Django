# Basic Product Management With Django

This is a Django-based Product Management System that allows users to manage products and categories with authentication-based roles.

---

## 🚀 Features
- User Authentication (Admin/Customer)
- Product Management (Add, Edit, Delete)
- Category Management
- Product Listing & Details View
- Role-based Access Control

---

## 🛠 Tech Stack
- Django (Python Web Framework)
- SQLite Database
- HTML, CSS (Frontend)

---

## 📌 Project Purpose
This project is built for learning Django CRUD operations, authentication, and relational database handling.
